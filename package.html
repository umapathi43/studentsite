iit-jee online test series

online test series - iit jee

A complete online assessment module for IIT-JEE Aspirants, the Online Test Series for IIT-JEE provides the students with flexibility of giving the tests at home and getting the score and performance analysis sitting at home.

All 60 tests come with a scientific performance analysis which compares student performance against the performance of IIT-JEE Aspirants across India. The performance analysis includes All India Rank, Percentile, Section-wise Rank and Percentile, IIT-JEE Score and more

online test series - jee advance

The Online Test Series includes:

  • Full Syllabus Taken Online- c total of 10 schedule based Full Syllabus
  • 60 Section wise tests taken Online.
  • All Self Administered Test through a Schedule
  • User interface is similar to actual Online IIT-JEE
  • Questions include numerical answer type
  • Detailed performance analysis including AIR, Score, Section wise analysis etc after each SATs

 

 

Fees for IIT - JEE ADVANCE : Rs 3500 .
